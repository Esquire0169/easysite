# EasySite

Многостраничный сайт сервиса EasySite (Web Giants) — сайты за 24 часа за 10 000 ₽.

## Стек

- Next.js (App Router) + React + TypeScript
- Tailwind CSS v4

## Команды

```bash
npm run dev
npm run lint
npm run build
```

## Расширение

- Новая страница: `app/<route>/page.tsx` + пункт в `lib/site.ts` → `navItems`
- Новые кейсы: `lib/cases.ts`
- Контакты и бренд-копирайт: `lib/site.ts`
